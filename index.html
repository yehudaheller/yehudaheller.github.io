Some Of My Projects

Neural Network for Detecting Pecking Behavior and Counting Chickens

I trained a neural network to detect pecking interactions between chickens in video footage and to count chickens standing on cooling perches. Initial experiments on a small video set were successful, and the project is ongoing pending a larger dataset.

Neural Network for Cow Head Position Analysis

I annotated a large dataset of cow images and trained a neural network capable of detecting the head, eyes (left/right), nose, and mouth (left/right). Using the trained model, I processed over 250,000 images to extract precise spatial keypoints for each cow.

Activity Reporting System for Engineers

I developed a full web-based system that enables research engineers to report daily or weekly activities and provides an admin interface for management. The system is currently in regular daily use.

Android Application for Microscopic Well Analysis

I developed an Android application that captures or uploads microscope plate images and automatically analyzes well intensities, similar to ImageJ. The app supports RGB, Luma, and Blue-Channel analysis and achieves accuracy comparable to ImageJ.
